## 🚨 Troubleshooting

### Canvas not showing?
- Check browser console for errors
- Verify Canvas API supported

### Furniture won't add?
- Check products.js loads
- Check console for errors

### Save not working?
- Verify localStorage enabled
- Try incognito mode

### Styles look off?
- Clear cache (Ctrl+Shift+R)
- Check CSS loaded
